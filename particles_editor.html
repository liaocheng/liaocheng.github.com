This is a gui editor for the particle system!

how to use :

step 1: open the editor,click import button.

step 2: open any file of example:

  • fire
  • circle
  • explode
  • snow
  • tail

copy body of the file to the code window of the editor.

step 3: click preview.

step 4: change any propertys of the example and then preview again.


you can download the air version editor from the project site
